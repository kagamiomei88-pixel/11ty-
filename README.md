# 11ty- 詞と沈黙の辞典

✴【澄響創世｜再勘】

詞と沈黙の辞典（核義記）— 名素・名胎・核義を記す非営利プロジェクト

## 概要

このプロジェクトは、言葉の音（澄響）と意味（核義）を記録し、再検討するための静的サイト生成プロジェクトです。11ty (Eleventy) を使用して構築されています。

## 特徴

- **澄響システム**: 各エントリーの音声情報（読み、発音、声調）を記録
- **核義記録**: 言葉の核となる意味を文書化
- **名素・名胎**: 言葉の要素と起源を追跡
- **タグ付け**: カテゴリーとテーマによる整理

## セットアップ

### 必要な環境

- Node.js 18以上
- npm または yarn

### インストール

```bash
# 依存関係のインストール
npm install
```

## 使い方

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:8080` を開いてサイトを確認できます。

### ビルド

```bash
npm run build
```

静的ファイルが `_site` ディレクトリに生成されます。

### クリーン

```bash
npm run clean
```

ビルド出力をクリアします。

## エントリーの追加

新しい辞書エントリーを追加するには、`src/entries/` ディレクトリに Markdown ファイルを作成します。

### エントリーのテンプレート

```markdown
---
layout: entry.njk
title: エントリーのタイトル
sound:
  reading: よみがな
  pronunciation: romaji
  tone: 声調
meaning: 短い意味の説明
tags:
  - タグ1
  - タグ2
---

## 名素

言葉の要素についての説明

## 語源

言葉の起源と歴史

## 用例

- 用例1
- 用例2
```

## プロジェクト構造

```
.
├── .eleventy.js          # 11ty設定ファイル
├── package.json          # プロジェクト設定
├── src/                  # ソースファイル
│   ├── _layouts/         # テンプレートレイアウト
│   │   ├── base.njk      # ベースレイアウト
│   │   └── entry.njk     # エントリーレイアウト
│   ├── _includes/        # 再利用可能なコンポーネント
│   ├── _data/            # サイトデータ
│   ├── entries/          # 辞書エントリー
│   ├── css/              # スタイルシート
│   │   └── style.css
│   ├── assets/           # 画像などの静的アセット
│   └── index.njk         # ホームページ
└── _site/                # ビルド出力（自動生成）
```
